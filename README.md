# Project Aegis

## Overview

Project Aegis is a long-term embedded systems engineering project focused on designing and developing a modular avionics development platform.

The objective is to learn professional embedded systems engineering through the design of hardware, firmware, communication interfaces, sensor integration, control systems, and data logging.

This project is intended to grow over multiple years, with each stage introducing new engineering concepts and increasingly sophisticated functionality.

---

## Long-Term Goals

- Read and process sensor data
- Estimate aircraft orientation
- Log telemetry
- Communicate with a ground station
- Implement control algorithms
- Drive actuators
- Design a custom PCB
- Follow professional software engineering practices
