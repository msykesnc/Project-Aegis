# 1. Purpose

This design review defines the mission, scope, and guiding philosophy of Project Aegis. The purpose of this document is to establish a clear foundation that will guide future engineering decisions throughout the project's development.

---

# 2. Background

Project Aegis is a long-term embedded systems engineering project inspired by modern aerospace and avionics systems. Rather than focusing on rapidly building a finished product, the project emphasizes developing professional engineering skills through deliberate system design, implementation, testing, and documentation.

The project will evolve over multiple versions, with each milestone introducing new embedded systems concepts while maintaining a cohesive system architecture.

---

# 3. Mission Statement

The mission of Project Aegis is to design, implement, and document a modular embedded avionics platform that develops professional embedded systems engineering skills through aerospace-inspired applications.

Project Aegis prioritizes engineering understanding over rapid development. Every subsystem, hardware component, and software module is selected to reinforce core embedded engineering principles while contributing to a cohesive, expandable system architecture.

The long-term objective is not only to produce a functional embedded platform, but also to create a professional engineering portfolio that demonstrates sound design practices, technical reasoning, and continuous learning.

---

# 4. Project Scope

### In Scope

- Embedded firmware development
- Professional software architecture
- Hardware interfacing and communication protocols
- Sensor integration
- System testing and validation
- Engineering documentation
- Modular hardware expansion
- Aerospace-inspired applications

### Out of Scope

- FAA-certified avionics
- Commercial product development
- Safety-critical flight software
- Autonomous aircraft development
- Features that do not contribute to the project's educational mission
