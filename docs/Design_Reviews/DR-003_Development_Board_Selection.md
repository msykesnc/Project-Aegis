# DR-003 — Development Board Selection

**Document ID:** DR-003
**Status:** Approved
**Date:** July 19, 2026
**Author:** Miller Sykes

## Purpose

Select the initial development board for Project Aegis.

## Decision

The STM32 NUCLEO-G474RE has been selected as the primary development board.

## Rationale

The board provides a professional development environment with integrated debugging, extensive peripherals, excellent documentation, and sufficient performance to support the long-term goals of Project Aegis.
